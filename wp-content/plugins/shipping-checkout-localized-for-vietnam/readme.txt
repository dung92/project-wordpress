== Description ==

Extended features for WooCommerce to match the market and the legality in Vietnam.

(Tiếng Việt: <a href="https://vi.wordpress.org/plugins/shipping-checkout-localized-for-vietnam">https://vi.wordpress.org/plugins/shipping-checkout-localized-for-vietnam/</a> )

### Features

* Support Vietnam administrative structure (municipality/province, district/county,ward/commune)
* Exchange rate when paying via Paypal  
* Customize fields of checkout page (set as not required, or hidden)
* Support a VAT box
* Change Vietnamese currency unit on the website  (đ » vnđ, đồng, etc)

### Extended features (Developing)

* VAT request (Customize display by the amount, example: > 200.000 vnđ)

== Installation ==

1. Download `shipping-checkout-localized-for-vietnam` using this link `/wp-content/plugins/` via FTP
2. Get `shipping-checkout-localized-for-vietnam.zip` directly on WordPress follows the 'Plugins->Add New' process in Dashboard.
3. Activate plugin then go to Settings if you want to use the features.
